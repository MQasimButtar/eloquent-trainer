🔒 Change Password

Your password hash is stored only in your browser — not in the app source code. Safe even on public GitHub.

⚠️ GitHub Security Note: Your public GitHub repo shows the source code, but the password hash in the code is just the default ("password"). Once you update via this form, your real hash is stored only in your browser's localStorage — invisible to anyone viewing the code. For full security, also set your GitHub repo to Private.
🔄 Sync Progress Across Devices

Same secret code on iPhone Safari + PC = synced progress. No servers, no account needed. Pure device-to-device transfer.

💡 How it works: Export a code on Device A → paste it on Device B. All sessions, writing drafts, quiz scores, and reading progress merge. Repeat whenever you want to sync.
🎙️ Voice Password: On the login screen, tap Try Voice Password and say "open eloquent" to unlock hands-free. Works in Chrome/Safari with microphone permission.

📱 Use Eloquent on iPhone — Free

Turn this web app into a full iPhone app using PWA (Progressive Web App) technology — completely free, no App Store needed

ℹ️
What is a PWA? A Progressive Web App lets you install any website to your iPhone home screen. It runs fullscreen like a native app, works offline for some features, and costs nothing. Eloquent is already PWA-ready.
Method 1 — Safari "Add to Home Screen" (Easiest)
1
Open Eloquent in Safari on your iPhone
Important: must be Safari, not Chrome or Firefox. Go to this URL in Safari:
Loading URL…
2
Tap the Share button
At the bottom of Safari, tap the square with an arrow pointing up (⬆) — the Share button.
3
Select "Add to Home Screen"
Scroll down in the share sheet and tap "Add to Home Screen". Give it a name like "Eloquent" and tap Add.
4
Launch from Home Screen
Eloquent now appears on your home screen like a real app. It opens fullscreen with no browser UI — just like a native app!
Method 2 — Host on GitHub Pages (Free, Permanent URL)
1
Create a free GitHub account
Go to github.com and sign up. It's completely free.
2
Create a new repository
Click "New repository", name it eloquent-trainer, make it Public, then create it.
3
Upload the HTML file
Rename the downloaded file to index.html. Upload it to the repo via "Add file → Upload files".
4
Enable GitHub Pages
Go to Settings → Pages → Source: "Deploy from branch" → Branch: main → /root → Save. Your URL will be:
https://yourusername.github.io/eloquent-trainer
5
Add to iPhone home screen
Open that URL in Safari on your iPhone, then follow Method 1 steps 2–4. Your keys and progress data save to the iPhone's localStorage.
Method 3 — Netlify Drop (30-second deploy)
1
Go to Netlify Drop
Open app.netlify.com/drop — no account needed.
2
Drag & drop the file
Rename your file to index.html. Drag it into the Netlify Drop page. It will deploy in under 30 seconds with a free HTTPS URL like:
https://random-name-123.netlify.app
3
Open in Safari → Add to Home Screen
Copy the Netlify URL, open in Safari on iPhone, then follow Method 1 steps 2–4. Done!
🎤 Microphone on iPhone — Important Notes

• Safari required — only Safari on iOS supports getUserMedia (microphone access)

• When you first tap 🎙️ Record, Safari will ask "Allow Eloquent to use your microphone" — tap Allow

• If you accidentally denied permission: Go to iPhone Settings → Safari → Microphone → Allow

• The Deepgram API key is stored in your iPhone's localStorage — it stays private on your device

• Progress data is saved locally — export it as JSON backup from the Progress page before clearing Safari cache

• For best audio quality, use iPhone's built-in mic or wired earphones in a quiet room

💡
Pro Tip: The fastest free option is Netlify Drop (Method 3). Just drag your file and get a shareable HTTPS URL in 30 seconds — no account needed. Then add to iPhone home screen via Safari.
